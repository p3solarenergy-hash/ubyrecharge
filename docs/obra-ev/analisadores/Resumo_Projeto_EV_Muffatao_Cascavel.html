Observações para Projeto do Carregador EV

Muffatão Cascavel — condições de projeto separadas do diagnóstico atual

Escopo deste documento

Este documento não descreve falhas existentes causadas pelo carregador. Ele registra cuidados de projeto caso o ponto receba um carregador EV de 60 kW.

O diagnóstico da instalação existente fica no resumo técnico separado. As recomendações abaixo dependem de projeto executivo, aprovação de concessionária quando aplicável e validação em campo.

Condições para implantação EV

Controle dinâmico de carga (DLM) Projeto EV

O carregador deve limitar automaticamente sua potência conforme a demanda instantânea do posto para preservar a capacidade da instalação.

Padrão e folga elétrica Projeto EV

O padrão de 200 A pode ser estudado com DLM, mas 250 A oferece maior margem operacional e deve ser avaliado na etapa de projeto/concessionária.

Circuito dedicado do carregador Projeto EV

Disjuntor, cabos, DPS, aterramento, seccionamento e queda de tensão devem ser definidos no projeto executivo do carregador.
