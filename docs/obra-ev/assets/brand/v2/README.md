# Identidade UBY Recharge

Biblioteca visual oficial usada pela plataforma.

## Uso na interface

- `01_principal_fundo_claro.png`: marca horizontal em fundos claros.
- `02_versao_dark.png`: marca horizontal em fundos escuros.
- `09_sobre_midnight.png`: simbolo para navegacao e paineis escuros.
- `14_icone_app.png`: favicon, PWA e atalhos.
- `17_marca_dagua.png`: marca d'agua discreta.
- `27_splash_loading.png`: estados de carregamento.
- `29_rail_dashboard.png`: referencia para navegacao lateral.
- `header_clube_uby.png`: cabecalho oficial do Clube UBY.

Os demais arquivos ficam preservados para aplicacoes institucionais, impressas
e promocionais. Cores, tipografia e regras de uso estao centralizadas em
`docs/obra-ev/brand.css`.
